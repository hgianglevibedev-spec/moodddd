# Moodmap Frontend Design Guide

This document outlines the design principles, color palette, and typography for the Moodmap web application.

## 1. Design Principles

*   **Playful and Modern:** The design should feel fresh, engaging, and fun.
*   **Clean and Intuitive:** The user interface should be easy to navigate and understand.
*   **Visually Driven:** Emphasize user-generated content like photos and maps.

## 2. Color Palette

*   **Primary Green:** `#c1ea6f`
*   **Primary Pink:** `#ff7cac`
*   **White:** `#ffffff`
*   **Gradient:** A linear gradient from `#c1ea6f` to `#ff7cac`.

## 3. Typography

### 3.1. Fonts

*   **H1 Headings:** Geomanist
*   **Subheadings (H2, H3, etc.):** Geomanist
*   **Body Text:** Roboto
*   **Functional Text (Buttons, Labels):** Roboto

### 3.2. Font Weights and Sizes

*   **H1:** Large and impactful. Size to be determined by context, but should be the most prominent text on the page.
*   **H2:** Bold (700 weight), large size.
*   **H3:** Semi-Bold (600 weight), medium-large size.
*   **Body:** Regular (400 weight), standard readable size (e.g., 16px).
*   **Functional Text:** Medium (500 weight), slightly smaller than body text.
